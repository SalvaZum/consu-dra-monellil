Términos del Servicio

1. Descripción del Servicio

Esta aplicación permite a los pacientes solicitar y gestionar turnos médicos con la Dra. Monelli, integrando el sistema con Google Calendar para recordatorios.

2. Uso Aceptable

El servicio debe usarse únicamente para gestionar turnos médicos reales. No se permite el uso fraudulento o malintencionado.

3. Integración con Google Calendar

La aplicación solicita acceso a Google Calendar exclusivamente para:

  • Crear recordatorios de turnos
  • Sincronizar horarios disponibles
  • Gestionar la agenda médica

4. Limitación de Responsabilidad

La aplicación es una herramienta de gestión y no reemplaza la consulta médica presencial cuando sea necesaria.
